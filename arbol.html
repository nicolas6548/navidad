Dia 2: Cómo construir un árbol de Navidad con SVG

Cómo usar el polygon elemento

Siempre podemos usar formas básicas para montar nuestras imágenes. A polygon es la forma más fácil de dibujar una forma de forma libre. Los polígonos tienen un points propiedad que establece una lista de coordenadas que están conectadas con líneas rectas.

Podemos la corona del árbol de tres triángulos. Empezamos con el de abajo, y lo agregamos capa por capa.

Aca tenemos el codigo para verlo

    <svg widht="200" height="400" viewBox="-100 -200 200 400">
      <polygon points="0,0 80,120 -80,120" fill="#234236"/>
    </svg>

Hicimos la parte de abajo del arbol, ahora vamos a hacer la segunda parte o la parte del medio con un color mas suave

Aca tenemos el codigo de la segunda parte para verlo

    <svg widht="200" height="400" viewBox="-100 -200 200 400">
      <polygon points="0,0 80,120 -80,120" fill="#234236"/>
      <polygon points="0,-40 60,60 -60,60" fill="#0C5C4C" />
    </svg>

Y para terminar vamos a hacer la parte de arriba del arbol

Con este codigo vemos terminado las 3 partes de arriba del arbol. Aca tenemos el codigo y nos faltaria solamente ponerle el tronco 

    <svg widht="200" height="400" viewBox="-100 -200 200 400">
      <polygon points="0,0 80,120 -80,120" fill="#234236"/>
      <polygon points="0,-40 60,60 -60,60" fill="#0C5C4C" />
      <polygon points="0,-80 40,0 -40,0" fill="#38755B" />
    </svg>

Por ultimo como dijimos mas arriba vamos a ponerle un tronco al arbol haciendo un rectangulo

Aca tenemos el codigo para verlo

    <svg widht="200" height="400" viewBox="-100 -200 200 400">
      <polygon points="0,0 80,120 -80,120" fill="#234236"/>
      <polygon points="0,-40 60,60 -60,60" fill="#0C5C4C" />
      <polygon points="0,-80 40,0 -40,0" fill="#38755B" />
      <rect x="-20" y="120" width="40" height="30" fill="brown" />
    </svg>
